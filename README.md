# Aesthetic Wallpapers

This repository contains a collection of aesthetic wallpapers that can be used for both personal and professional use.

## Typing Animation

To implement typing animations in your project, update the URLs to:

```javascript
const typingAnimation = new TypingAnimation({
    lines: ["Hello, World!", "Welcome to Aesthetic Wallpapers!"],
    color: "A855F7",
});
```

- Ensure that the `lines` parameter does not contain pipe characters, and all strings are properly formatted.

Enjoy your beautiful wallpapers!
